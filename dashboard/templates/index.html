How it works ?

Our models have been trained on the IMDB Large Movie Review dataset and tested on the Amazon Movie Reviews dataset to perform the specific task of predicting the class of a movie review. To use, you can select the model you want to use and then import an excel or csv file containing all your reviews. Then the model will analyze them and return meaningful statistics and the classification results
